# SageNano

A terminal text editor inspired by nano, written in [SageLang](https://github.com/Night-Traders-Dev/SageLang).

## Installation
`sagepkg install SageNano`

## Usage
`SageNano [filename]`

## Key Bindings
- `^X`: Exit
- `^O`: Save (Write Out)
- `^W`: Search
- `^K`: Cut Text
- `^U`: Uncut Text
- `^Y`: Previous Page
- `^V`: Next Page
- `^G`: Help

## Features
- **📝 Basic Editing**: Smooth text insertion and deletion.
- **📄 Paging**: Fast navigation with Page Up/Down keys.
- **🔍 Search**: Find text within the buffer.
- **✂️ Cut & Paste**: Efficient line-based cut/uncut functionality.
- **🛠️ Built-in UI**: Clean terminal interface with status and shortcut bars.

## Project Structure
- `universal/`: Source code.
