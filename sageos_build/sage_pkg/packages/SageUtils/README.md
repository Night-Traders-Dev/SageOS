# SageUtils

Common terminal utilities implemented in [SageLang](https://github.com/Night-Traders-Dev/SageLang).

## Installation
`sagepkg install SageUtils`

## Included Utilities
- **cat**: Concatenate and print files.
- **ls**: List directory contents.
- **mkdir**: Make directories.
- **rm**: Remove files.
- **echo**: Print text to stdout.

## Features
- **🚀 Native Implementation**: Fast and lightweight alternatives to standard system tools.
- **🛠️ Cross-platform**: Works wherever SageLang runs.

## Project Structure
- `universal/`: Source code.
- `metadata.json`: Package metadata.
