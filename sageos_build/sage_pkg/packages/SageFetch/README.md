# SageFetch

A Neofetch clone in Sage.

## Installation
`sagepkg install SageFetch`

## Usage
Run `SageFetch` to display system information.

## Features
- **🖼️ ASCII Art**: Custom Sage-themed logo.
- **🖥️ System Info**: Displays OS, Kernel, Uptime, Shell, CPU, and Memory.
- **🔍 Robust Detection**: Intelligent CPU identification (ARM/x86) and accurate shell fallback detection.
- **🚀 Speed**: Written in pure Sage for high performance.

## Project Structure
- `universal/`: Source code.
- `aarch64/`: Native binaries for ARM64.
- `x86_64/`: Native binaries for x64.
